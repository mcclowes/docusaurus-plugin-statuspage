# Docusaurus Glossary

Use when working with docusaurus-plugin-glossary to configure, manage glossary terms, troubleshoot issues, and explain features.

## What This Skill Covers

- Plugin configuration in docusaurus.config.js
- Creating and managing glossary.json files
- Using automatic term detection and manual GlossaryTerm component
- Troubleshooting common issues (404s, missing tooltips, auto-linking problems)
- Customizing styles and components

## Structure

- `SKILL.md` - Quick reference for common tasks
- `references/configuration.md` - Plugin options and setup details
- `references/usage.md` - Using terms, components, and customization
- `references/troubleshooting.md` - Common issues and solutions

## Usage

This skill is automatically discovered by Claude when working with docusaurus-plugin-glossary. Invoke explicitly with `/skill docusaurus-glossary` if needed.
